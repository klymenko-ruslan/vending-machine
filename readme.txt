Run application using:
mvn spring-boot:run

Application will start work on localhost:8080. There are two accessible web services:
http://localhost:8080/optimal-change/{pence}
http://localhost:8080/change/{pence}

 Optimal-change calls getOptimalChangeFor implementation.
 Change calls getChangeFor implementation.
